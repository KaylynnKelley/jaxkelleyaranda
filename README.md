# Jax Kelley-Aranda — Soccer Recruiting Site

College soccer recruiting profile for Jax Kelley-Aranda (Class of 2028, Crossfire Premier XF Academy B10 ECNL).

Live site: https://jaxkelleyaranda.com

## What's here
- `index.html` — the entire site (self-contained: all photos and styles embedded)
- `CNAME` — tells GitHub Pages to serve the site at jaxkelleyaranda.com
- `.nojekyll` — disables Jekyll processing (serves files as-is)
- `robots.txt` / `sitemap.xml` — basic SEO so coaches can find the site via search

## How to update
- **Highlight reel:** open `index.html`, find the YouTube embed URLs, swap in the new video ID. Or replace the video on YouTube keeping the same link — the site updates itself.
- **Anything else:** edit `index.html` and re-upload (commit) it to this repo. The live site refreshes within a minute or two.
